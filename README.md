# Dina GitHub Portfolio

Simple static portfolio website for GitHub Pages.

## Files
- `index.html` — website structure
- `style.css` — design
- `script.js` — scroll animations
- `images/` — folder for your own images

## How to publish on GitHub Pages
1. Create a GitHub repository.
2. Upload all files from this folder.
3. Go to Settings → Pages.
4. Choose `Deploy from a branch`.
5. Select `main` and `/root`.
6. Save and wait for the website link.

## What to edit
In `index.html`, change:
- `yourusername` to your Telegram username
- `your@email.com` to your email
- project names, texts and prices
